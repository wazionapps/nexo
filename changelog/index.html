Release History

The last releases that actually changed the runtime

Short, public-facing highlights. The full engineering history remains in the repository changelog.

2.6.5 · 2026-04-04

Safer wake/update behavior and public surface cleanup

Clarified power-helper semantics, suppressed duplicate catch-up relaunches for in-flight cron windows, and made runtime updates reconcile declared personal schedules automatically.

2.6.3 · 2026-04-04

Fix runtime update gaps

Fixed same-file sync issues during runtime update, normalized Claude Code hook migration, and realigned packaged runtime version detection.

2.6.2 · 2026-04-04

Startup preflight, power policy, personal recovery

Added startup preflight before chat/server startup, persisted the optional power policy, and brought managed personal schedules into the wake/catch-up recovery path.

2.6.0 · 2026-04-03

Personal scripts become first-class

Introduced the personal scripts registry, schedule lifecycle, plugin structure for Claude Code, and removed the old day orchestrator from core.

2.5.0 · 2026-04-03

Runtime CLI and unified doctor

Shipped the nexo CLI, modular diagnostics, runtime update flow, executable skills, and operational entrypoints that define the modern product surface.
