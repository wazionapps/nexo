Tool Surface

150+ MCP tools, one runtime

NEXO does not hide its capabilities behind one giant black box. The runtime exposes memory, diagnostics, updates, graph, automation, and scripting as explicit tools.

Sessions

Startup, heartbeat, context continuity

Launch, keep state warm, recover after compaction, and hand structured context to delegated work.

Memory

Recall, learnings, reminders, episodes

Persistent memory spans semantic recall, followups, reminders, changes, session diaries, and long-term learnings.

Safety

Guardrails, trust, cognitive checks

Before edits or risky action, the runtime can surface prior mistakes, contradictions, and missing evidence.

Operations

Doctor, update, backup, schedule

The MCP layer also exposes runtime operations, not just memory retrieval, so the agent can inspect and repair its own environment.

Personal automation

Scripts, reconcile, scheduling, recovery

Your own scripts can be classified, scheduled, reconciled, and given wake-aware recovery policies without manual plist drift.

Knowledge

Graph, entities, artifacts, skills

NEXO also models entities, relationships, reusable skills, and artifact inventory so the agent can reason structurally, not just semantically.
