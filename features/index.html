Deep Dive

How NEXO Brain Works

A detailed walkthrough of the cognitive architecture, memory model, and autonomous processes.

Autonomous

13 core jobs plus optional helpers, zero babysitting

NEXO Brain runs a core set of recovery-aware jobs that maintain memory health, generate intelligence briefs, and self-repair. Optional helpers can be layered on top depending on platform and user configuration.

Cognitive Decay

Daily Ebbinghaus memory sweep. Weakens unrehearsed memories and promotes surviving STM items to LTM.

Sleep System

Nightly memory cleanup and pruning. Consolidates the day's learning and removes noise while the system is idle.

Daily Synthesis

Morning intelligence brief. Summarizes yesterday's errors, patterns, pending items, and context for the day ahead.

Self-Audit

Daily behavioral self-evaluation. Checks for repeated errors, ignored learnings, and drift from the operational codex.

Post-Mortem Consolidator

Extracts learning patterns from session histories. Turns individual incidents into reusable prevention rules.

Immune System

Health monitor and auto-repair. Detects database corruption, stale locks, and process failures, then fixes them.

Deep Sleep

Reads Claude Code and Codex transcripts overnight, then blends recent and older context across a 60-day window, weights projects by leverage signals, and writes daily, weekly, and monthly summary artifacts.

Catch-Up

Recovers missed tasks on Mac boot. If the machine was asleep during scheduled processes, catch-up runs them all.

Auto-Update

Checks for new versions and applies updates. Keeps the cognitive engine current without manual intervention.

Followup Hygiene

Automatically cleans stale followups, closes completed ones, and ensures no deferred tasks are forgotten. Runs nightly.

GitHub Monitor

Daily check for open issues, PRs without review, and unreleased commits on the NEXO Brain repository. Alerts on actionable items.

New in v2.4.0

Skill Auto-Creation

Deep Sleep extracts reusable procedures from sessions. Steps, gotchas, and markdown content stored automatically. Trust pipeline controls quality.

Cron Scheduler

Every cron execution tracked in cron_runs table. nexo_schedule_status shows overnight health. nexo_schedule_add creates new crons from conversation.

Deep Sleep Watermark

Tracks last processed timestamp instead of filtering by date. Late-night sessions never missed. Per-session checkpointing survives crashes.

Security Hardening

Credential redaction in tool logs, transcript sanitization, command injection fix, path traversal protection. Passed 5-phase automated audit.

Hooks

Claude Code hooks and client-specific integration depth

Claude Code still has the deepest hook integration in NEXO. Codex now compensates much more strongly through managed config, startup discipline, and runtime doctor audits, while Claude Desktop remains an explicit MCP-only shared-brain companion.

session-start

Generates a briefing with pending tasks, recent session context, and morning intelligence. Your agent starts every session informed.

session-stop

Captures session state, writes the session diary, and logs mental state for continuity. No context lost between sessions.

capture-session

Sensory Register layer (Atkinson-Shiffrin Layer 1). Captures raw session input for attention filtering and memory promotion.

caffeinate-guard

Keeps the Mac awake during nocturnal processes. Ensures cognitive decay, sleep system, and synthesis run to completion.
