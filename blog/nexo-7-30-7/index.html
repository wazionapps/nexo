NEXO 7.30.7 - Deep Sleep retention closeout

v7.30.7 is a micro-release over v7.30.6. It carries the same Deep Sleep retention behavior, but republishes the package with the missing version-specific smoke artifact and release contract included in the tagged tree.

The runtime behavior remains focused: Deep Sleep keeps the newest database backups and analyzed context dumps, preserves unfinished contexts, rotates oversized sleep logs, and applies the policy during update for existing installs.

This release exists to keep the public release process honest. v7.30.6 reached npm, but final closeout detected that the required release-contracts/smoke/v7.30.6.json artifact was not present in the tag. v7.30.7 fixes that packaging/evidence gap instead of masking it locally.

python3 scripts/verify_release_readiness.py --ci --contract release-contracts/v7.30.7.json --require-contract-complete --require-smoke
python3 scripts/verify_release_readiness.py --final-closeout --protocol-task-id PT-1780558689-16833
