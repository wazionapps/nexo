NEXO 7.27.2 - OpenAI provider migration hotfix

v7.27.2 is a focused Desktop-managed Brain patch for provider-account migration. Older installs that selected Codex before the provider-runtime schema now keep launching OpenAI/Codex after Desktop refreshes Brain.

The F0.6 config-folder transition also keeps reading the existing config/schedule.json until it has been migrated, so creating personal/config/ no longer hides a user's selected provider.

This release is bundled by NEXO Desktop 0.41.3 so existing 0.41.2 users receive a real newer Brain refresh with the migration fix.

nexo provider status
nexo provider select openai
nexo preferences --chat-provider openai --automation-provider openai
