NEXO 7.23.4 - release publish verification

v7.23.4 is a Brain release-engineering patch. The publish workflow now stops on real npm publication failures instead of turning every failed publish into a misleading "Version already exists" message.

The release also keeps OpenClaw package-lock metadata synchronized with the public package version, so package.json, lockfile, ClawHub metadata and the MCP bridge version move together.
