NEXO 7.28.0 - cognitive memory architecture

v7.28.0 is a Brain minor release focused on memory quality. The runtime now has explicit structures for action authorship, reasons, operational state, runtime budget, entity live profiles, failure prevention, semantic layers, and memory utility so future answers can connect what happened, why it happened, and what should be reused.

The release adds a Memory Executive and a runtime memory benchmark gate covering ten cognitive specs. That gate is designed to catch regressions where memory data exists but is not actually usable before an answer.

Pre-answer routing also stays safer and faster: hot paths avoid heavy transcript fallback, semantic caches reduce keyword-only matching, and export/context surfaces redact sensitive payloads, raw snippets, paths, and private values by default.

Release readiness for v7.28.0 requires both a clean schema-76 fresh install and an update path from schema 75 to 76 that preserves existing sessions and diary rows while adding semantic-layer data.

python3 scripts/verify_release_readiness.py --ci --contract release-contracts/v7.28.0.json --require-smoke --require-contract-complete
scripts/pre-release-verify.sh --release v7.28.0
