NEXO 7.27.1 - provider lifecycle hotfix

v7.27.1 is a focused Desktop-managed Brain patch for provider-account sessions. Lifecycle stop calls now skip external Anthropic/OpenAI child-session UUIDs safely instead of treating them as internal Brain session IDs.

The provider runtime also keeps chat and automation aligned to the same selected Anthropic/OpenAI account. If no verified account remains active, background automation fails closed instead of silently trying a stale provider.

This release is bundled by NEXO Desktop 0.41.2 so existing 0.41.1 users receive a real newer update with the provider lifecycle and account-selection fixes.

nexo provider status
nexo provider select openai
nexo preferences --chat-provider openai --automation-provider openai
