NEXO 7.37.3 - release pipeline timeout hardening

v7.37.3 is the public release line for the 7.37.2 runtime hardening plus one release-pipeline fix. The v7.37.2 tag attempt did not publish npm or GitHub release artifacts because the publish workflow test job timed out while pytest was still progressing.

The full test gate stays in place. The publish workflow still runs the full Brain pytest suite before public-channel publication. The change is the budget: slow runners now have enough time to finish the suite and release readiness.

No partial public Brain release was shipped from v7.37.2. The downstream publish job was skipped after the timeout, so v7.37.3 is the version users and installers should consume.
