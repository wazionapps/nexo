NEXO 7.30.9: release-safe self-heal

June 4, 2026. This patch makes the update path and release closeout judge the current repaired installation instead of stale local history.

What changed

  • Post-update repair now writes a verified baseline after runtime sync and import verification.
  • Doctor checks distinguish installed-product failures from operator/session history.
  • Release closeout blocks real installation failures, but keeps old Codex/session drift as advisory history.

Why it matters

Users should not need to run manual commands after updating. If NEXO can safely repair bootstrap, config, hooks, LaunchAgents, and runtime state, the update should do it and then evaluate the system from that repair point forward.
