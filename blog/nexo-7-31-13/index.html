NEXO 7.31.13 - offline wheel bundle no longer hard-pins onnxruntime

v7.31.13 is a patch release over v7.31.12. It fixes the offline dependency bundle so it resolves correctly across every target platform.

7.31.12 hard-pinned onnxruntime==1.26.0. That wheel only ships for newer manylinux (2_28) and modern macOS, but the offline wheel bundle also targets older manylinux tags (2_17 / 2014) for broad Linux/WSL compatibility, and those cap at 1.19.2. A single == pin could not be satisfied for every target, which broke both the cross-platform wheel bundle and Brain installs on older Linux.

onnxruntime (fastembed's native transitive dependency) is now left unpinned, so fastembed/pip resolve the best onnxruntime wheel per platform. fastembed==0.8.0 stays hard-pinned because it is pure-python and available everywhere.
