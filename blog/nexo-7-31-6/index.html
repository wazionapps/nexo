NEXO 7.31.6: Desktop UI language for headless notices

June 11, 2026. A small patch for language consistency across Desktop-driven automation.

What changed

  • Headless/email-monitor notifications now respect the Desktop UI language. Static emails for needs_interactive, provider breaker pause, and provider breaker resume choose explicit ES/EN templates from Desktop app.ui_language first, then fall back to profile/calibration language and English.
  • Templates stay product-generic. They use the configured user and assistant names, so releases do not bake in one operator or one agent identity.
