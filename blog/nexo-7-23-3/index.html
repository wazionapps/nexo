NEXO 7.23.3 - followup runner DONE filtering

v7.23.3 is a small Brain patch for followup automation hygiene. The followup runner executable-batch query now treats DONE as terminal alongside BLOCKED, ARCHIVED, DELETED and WAITING.

The practical fix is that already-finished followups stay out of automation queues instead of being picked up again during background execution.
