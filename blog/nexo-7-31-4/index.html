NEXO 7.31.4: memory recall honours absolute time ranges

June 11, 2026. A focused patch on the Brain's recall tools, straight from an operator repro.

What changed

  • Absolute time ranges work. time_range only understood relative values (today, yesterday, last N days); an ISO date, a start..end range, a datetime or epoch seconds silently disabled the filter, so asking about a specific past day returned the most recent events instead. All those forms now parse.
  • The window is enforced in SQL. Filtering used to happen in Python after a recency-truncated fetch, so old windows could never surface. The fetch helpers now take real time bounds.
