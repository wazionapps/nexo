NEXO 7.23.11 - packaged update compatibility

v7.23.11 is a Brain update-compatibility patch. Older installed runtimes can now import the newer updater even before cognitive_paths.py has been synced into place.

This lets the package sync finish, deliver the cognitive DB cleanup module, and move users past legacy cognitive.db layouts without blocking the update path.
