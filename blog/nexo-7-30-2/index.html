NEXO 7.30.2 - Deep Sleep release-grade overnight loop

v7.30.2 is a Brain patch release focused on making Deep Sleep operational instead of ceremonial. The nightly wrapper now fails closed when extraction, synthesis, application, or audit phases fail, so a partial run cannot update the watermark as if the night had been processed correctly.

The sleep session now writes complete learning context coverage through a full dump plus chunk files, records coverage metadata, and exposes startup health warnings when synthesis quality is not good enough. That gives the next session explicit evidence of what happened overnight instead of relying on a truncated prompt.

Deep Sleep synthesis also emits an agent start packet with the prioritized agenda, context packets, review items, and counts. Startup prefers that packet before falling back to older synthesis files, so the next Nero session can begin with the right context immediately.

Finally, followups created by Deep Sleep now carry practical governance fields: date, verification text, priority, owner, and internal flag. The result is a loop that can turn overnight findings into reviewable work with release-pinned tests.
