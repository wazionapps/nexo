NEXO 7.30.1 - morning briefing clarity

v7.30.1 is a Brain patch release focused on the daily morning briefing experience. The briefing is now framed as a start-of-day assistant: it should filter noise, rank what matters, connect related items, and help the operator understand what needs attention first.

The release also clarifies optional external sources. News and weather are included only when verified collected data exists, and unavailable sources must never be invented or allowed to block the rest of the briefing.

NEXO no longer asks the operator to choose a manual user type for the morning briefing. It infers the useful angle from profile fields, recent activity, explicit preferences, and available context, while staying general and practical when context is thin.

This patch coordinates with NEXO Desktop 0.43.1, where the content preference modal now shows real help tooltips and explains what activating News or Weather will do.
