NEXO 7.23.2 - fresh Desktop Brain update detection

v7.23.2 is a small Brain patch for coordinated Desktop releases. Desktop consumes `nexo --version --json`; that path now refreshes npm when the cached latest version is missing, equal, or older than the installed Brain version.

The practical fix is that a Brain release published minutes after a user checked versions can still appear in NEXO Desktop instead of being hidden by a stale local latest-version cache.

scripts/pre-release-verify.sh --release v7.23.2
