NEXO 7.25.0 - Memory Fabric

v7.25.0 turns disconnected memory islands into one repairable fabric. Transcript metadata, historical backup diaries, unified search and knowledge graph evidence now reinforce each other instead of relying on a slow raw scan at answer time.

The backup retention path now reconciles recoverable diary rows before hourly or weekly DB snapshots can be deleted. Historical diaries are indexed once with stable content hashes, surfaced through unified search and linked to sessions, areas and email evidence in the knowledge graph.

Existing installs get the same repair path through update and doctor. The release also refreshes stale transcript indexes, supports exact and prefix transcript ids, preserves same-basename Codex sessions across active/archive roots, and widens pre-answer transcript lookup to the product's long-horizon window.
