NEXO 7.30.20: packaged product knowledge hotfix

June 6, 2026. This patch fixes the packaged update path for the product knowledge hub introduced in v7.30.19.

What changed

  • Packaged installs now copy product_knowledge/ alongside core runtime packages such as local_context/, doctor/, db/ and cognitive/.
  • nexo update backup, promotion, shim and restore paths now preserve the product knowledge package.
  • Installer and update regression tests assert that runtime package directories include product_knowledge.

Why it matters

v7.30.19 added the structured catalog and read-only product capability tools. v7.30.20 makes that contract installable from the npm package, so a real packaged runtime can import the new tools immediately after update.
