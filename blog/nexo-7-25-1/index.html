NEXO 7.25.1 - operational guardrail audit cleanup

v7.25.1 is a focused operational patch after the Memory Fabric release. It keeps the guardrail audit useful by removing noisy false positives and making review queues explicit by severity.

The shell guardrail extractor now skips non-path arguments for common HTTP flags such as user agents, headers, cookies, data payloads, forms and request methods. Version-like fragments lifted from strings such as Mozilla/5.0 or AppleWebKit/537.36 are filtered before they can become fake file paths.

The protocol-debt drain now groups review-needed items by severity and debt type. Daily self-audit output can show ERROR, WARN and other buckets without hiding new error classes behind a fixed short summary.
