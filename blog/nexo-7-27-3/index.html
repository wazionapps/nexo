NEXO 7.27.3 - OpenAI system-role compatibility hotfix

v7.27.3 is a focused compatibility patch for Desktop-managed OpenAI accounts. OpenAI raw calls now fold runtime instructions into the user payload instead of sending a separate system-role message.

Codex-managed bootstrap also stops writing legacy initial_messages, and doctor now flags old Codex configs that still contain them so a refresh can repair the machine before a user sees an API model error.

This release is bundled by NEXO Desktop 0.41.4 so the app waits for the active Brain version to settle before letting the user continue after a refresh.
