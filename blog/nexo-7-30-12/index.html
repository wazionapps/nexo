NEXO 7.30.12: Desktop-managed email credentials

June 4, 2026. This patch prepares the Brain email layer for the managed Email NEXO assistant-account flow in Desktop.

What changed

  • Email account reads, config loading, and migration now share a single credential helper.
  • The helper resolves keychain-backed credential markers first, then falls back to legacy SQLite secrets for older installs.
  • Tests now cover the keyring marker path so Desktop-managed email setup can migrate without exposing flat secret files.

Why it matters

Email NEXO provisions a real IMAP/SMTP assistant mailbox from the backend, while Brain still has to read and use configured email accounts locally. 7.30.12 keeps that boundary clean: Desktop and migration code can store credentials through the managed credential service, and Brain reads them through one predictable helper.
