NEXO 7.30.23: Opportunity Orchestrator Phase 1

June 7, 2026. This patch adds the first Brain-owned anticipatory layer: it looks at real operational signals, prepares safe context ahead of the user, and surfaces only scarce, evidence-backed proposals.

What changed

  • Migration 80 adds durable signals, opportunities, evidence, preparations, proposals, proposal events, suppression rules and action authorizations.
  • The new orchestrator can refresh opportunities, queue at most three normal proposals, return evidence, accept feedback and suppress repeated noise.
  • Dry-run JSON and Markdown reports can be written without sending messages, touching production or executing external actions.
  • Release tests cover zero-proposal days, max-three output, evidence requirements, feedback/suppression, authorization boundaries and visible-output privacy guardrails.

Why it matters

NEXO should get ahead of work without becoming noisy. This release gives Brain a small, auditable queue that can prepare context before the user asks while still treating no proposal as a healthy result when nothing has enough evidence.
