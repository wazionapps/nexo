NEXO 7.31.0: default Claude model Fable 5

June 10, 2026. This minor release moves the recommended Claude Code model from Opus 4.8 to Fable 5 with max reasoning (claude-fable-5).

What changed

  • All four main resonance tiers now resolve Claude Code to Fable 5. The muy_bajo tier keeps Haiku for cheap internal classifiers, and Codex stays on GPT-5.5.
  • Existing installs riding NEXO defaults auto-migrate on update, mirroring the 4.6→4.7→4.8 migrations, while customized models are respected.
  • Learning housekeeping no longer aborts when the embedding backend is missing: dedup degrades to a skip instead of crashing the whole decay/prioritization/archival run.

Why it matters

Operators riding NEXO defaults should always get the current recommended Claude Code model without manual reconfiguration, while anyone who deliberately pinned a different model keeps their choice untouched across updates.
