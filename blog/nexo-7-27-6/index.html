NEXO 7.27.6 - operational memory continuity

v7.27.6 is a focused Brain patch for operational memory. Detected future-action promises now persist as commitments with session, evidence, dedupe, and action-link fields instead of living only in a short-lived enforcement window.

Pre-answer routing now handles operator continuity questions such as qué prometí?, qué queda pendiente?, and por qué toqué src/pre_answer_router.py through evidence-backed sources before falling back to transcripts. The cognitive source now reads real Memory Observations through memory_search, preserving evidence references.

Memory health now reports an observation-queue SLA and fails when old pending rows prove captured events are not becoming searchable observations. The saved-not-used audit also measures Local Context usage from local_context_usage_events.used_before_response rather than the legacy query counter.
