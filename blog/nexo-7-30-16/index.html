NEXO 7.30.16: embedding migration diagnostics

June 5, 2026. This patch aligns the Brain runtime with the coordinated Desktop model-update release.

What changed

  • nexo_embedding_migration_status reports migration status, progress, errors and backup path without warming models or downloading files.
  • Migration coverage now pins resumable shadow embeddings and read-only status behavior for Desktop diagnostics.
  • Release checks cover bundled model verification and obsolete managed model cleanup for packaged Desktop updates.

Why it matters

Existing installs need safe visibility while memory embeddings migrate to the bundled local model stack. 7.30.16 lets Desktop detect migration health without triggering downloads or changing runtime state, and keeps update cleanup covered by tests.
