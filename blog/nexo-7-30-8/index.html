NEXO 7.30.8 - Deep Sleep and Local Context stability fixes

v7.30.8 is a patch release over v7.30.7. It fixes two operational issues found during closeout: inflated Deep Sleep session counts from parallel Codex sub-agents, and runaway Local Context entity_facts growth caused by attributing every chunk candidate to every entity in a file.

Deep Sleep now keeps sub-agent content but folds spawned explorers into the parent work thread before counting and analyzing sessions. The 2026-06-04 run that looked like 11 sessions resolves to 4 work threads with no transcript content lost.

Local Context now writes facts only for entities actually mentioned in the same chunk, caps facts per asset, and drops paragraph-length values that are not useful facts. This release deploys the code fix; deleting already-written oversized rows remains a separate operational purge.
