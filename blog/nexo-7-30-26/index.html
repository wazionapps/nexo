NEXO 7.30.26: F0.6 core script propagation

June 8, 2026. This patch makes runtime update copy packaged core scripts into the active F0.6 code path. It ensures fixes like the watchdog maintenance cleanup in v7.30.25 reach the file LaunchAgent actually executes.

What changed

  • nexo update now preserves the F0.6 scripts -> core/scripts compatibility shim and copies packaged scripts to the resolved core directory.
  • Core scripts are no longer blocked by stale non-script classification from an old registry scan.
  • Personal script collisions are still preserved, so operator-owned scripts are not overwritten.
  • Regression tests cover the exact active path used by watchdog LaunchAgents after update.

Why it matters

A release can publish the right package and still fail users if the updater preserves an old active runtime script. This patch closes that gap for F0.6 installs.
