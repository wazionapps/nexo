NEXO 7.20.8 — Windows Mail and Outlook Mac roots

Published 2026-05-13. Patch release over v7.20.7.

Why this release exists

Local email coverage must work across normal consumer and business setups. v7.20.8 closes a remaining root-classification gap where Windows Mail package directories and Outlook Mac profile directories could be rejected as generic private app data before the bounded email allowlist had a chance to apply.

What changed

  • Windows Mail roots are recognised. `microsoft.windowscommunicationsapps_*` LocalState trees are treated as local email sources, allowing supported `.eml` files to be indexed.
  • Outlook Mac roots are recognised. Existing Outlook profile roots under `Library/Group Containers/UBF8T346G9.Office/Outlook` are no longer blocked as generic Group Containers.
  • The safe allowlist still applies. Brain only extracts supported email-like files and keeps large or proprietary stores bounded/inventory-only where no safe parser exists.
