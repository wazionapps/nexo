NEXO 7.30.22: managed MCPs and closure plane

June 7, 2026. This patch coordinates Brain 7.30.22 with NEXO Desktop 0.43.14 and ships the same managed MCP and closure-plane payload under a fresh runtime version so packaged updates cannot skip the new managed_mcp package.

What changed

  • Brain now owns a managed default MCP catalog and exact lockfile for Chrome control, desktop control and power-control providers.
  • Release prep now verifies managed MCP provider locks against npm latest, including version, integrity, tarball and MCP binary selection.
  • The Operational Closure Plane adds canonical closure items, sources, events and verification tooling on top of protocol tasks, followups, outcomes, protocol debt and the MCP write queue.
  • NEXO Desktop 0.43.14 consumes the coordinated Brain contract and starts the Active Turn Input Router path with Claude stream-json support and Codex app-server fallback planning.

Why it matters

NEXO can now keep its default MCP capabilities in sync through release automation instead of manual memory, while the closure plane gives open operational work a single ranked, evidence-backed place to converge.
