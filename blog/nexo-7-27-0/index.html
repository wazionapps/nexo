NEXO 7.27.0 - Anthropic/OpenAI account parity

v7.27.0 closes the provider-account parity work for Desktop-managed Brain. Anthropic and OpenAI now share the same model-default contract, runtime sync path, managed install recovery, and provider metadata behavior.

The OpenAI path now uses GPT-5.5 with mapped reasoning effort levels, while Anthropic stays on the current Opus 4.7 profile. Background automations and crons pick up the selected provider at restart instead of keeping a stale runtime account.

Release checks also cover LaunchAgent template paths, provider runtime recovery, and public copy so Desktop and Brain speak in terms of Anthropic/OpenAI provider accounts rather than mixing engine names into user-facing surfaces.
