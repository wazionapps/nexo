NEXO 7.33.0 - Cognitive OS Ola 1 phase 2: recall by meaning + graph at answer time

v7.33.0 is the second wave of the "NEXO Cognitive OS" plan. It makes recall work the way a brain does - by meaning, not just exact words - and finally reads the knowledge graph at the moment you ask.

Memory search finds by meaning. Observations are embedded at write time and search fuses full-text with vector similarity, so a paraphrased question (sharing no exact word with the stored memory) still retrieves it. It degrades safely to full-text when no embedding model is loaded, and never embeds on a cold model on the answer path.

The graph is read at answer time. A bounded, fail-open kg_neighbors source surfaces related entities, files and verified causal edges for whatever you mention - the read side of the connect-the-dots substrate v7.32.0 started writing.

Local files by meaning-keyword (FTS5), reliable nightly consolidation (no more timeouts), reliable correction capture (soft), and an atomic runner lock round out the release. All additive, reversible, and verified against the full test suite.
