NEXO 7.20.7 — CI-safe local email roots

Published 2026-05-13. Patch release over v7.20.6.

Why this release exists

v7.20.6 added safe local email roots for Mail.app, Outlook, Thunderbird and NEXO email continuity. The full Linux CI suite exposed a deterministic test gap: a migrated or simulated macOS mail store could exist while the process was not running on macOS, so the root bootstrap needed to include existing platform-specific stores instead of relying only on the current OS branch.

What changed

  • Email root bootstrap is deterministic. Brain keeps native platform roots and also includes platform-specific mail stores that already exist under the user profile.
  • CI, WSL and migrated profiles are covered. The same local email coverage contract now holds when tests or host environments expose Mail.app / Outlook-like trees from another platform.
  • No privacy broadening. The bounded allowlist remains in place: supported email files and NEXO email DB continuity are indexed, while private profile folders and credential-like files stay blocked.
